# Frozen Station Splits

These files define the station-disjoint roles used in the manuscript:

- `train_val_stations.txt`: 15 training/internal-validation stations.
- `development_stations.txt`: 5 stations used for frozen checkpoint selection.
- `final_test_stations.txt`: 34 held-out final event-template stations.

The 54-station split keeps the Raspberry Shake AM sensor family fixed while
spanning geographically diverse site and cultural-noise environments.
