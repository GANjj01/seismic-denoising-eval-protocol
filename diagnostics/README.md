# Diagnostics

- `noise_correlation_diagnostic_summary.csv`: white-noise and finite-Welch
  opportunity baselines plus station-level noise autocorrelation/coherence
  summaries.
- `noise_correlation_by_station.csv`: station-level diagnostic rows.
- `covnorm_region_consistency.csv`: broad-region check showing that the
  CovNorm Z-correlation loss is not driven by a single geographic cluster.
